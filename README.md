# demo-graphql-codegen-vue

## 事前準備

### API 編

Prisma x GraphQL で簡単に API を導入する

[API server](https://github.com/shippokun/prisma-graphql-apollo-server)

### Client 編

依存関係のインストール

```bash
npm install
```

## 開発

```bash
npm run start
```

Run your unit tests

```bash
npm run test
```

codegen

```bash
# 事前に API server を起動していること
# Client codegen command
npm run generate
```
